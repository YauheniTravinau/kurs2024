Для чего нам нужно уметь создавать "html" с помощью JS?

Создание HTML с помощью JavaScript позволяет динамически изменять содержимое веб-страницы в ответ на различные события или условия.
Что такое DOM?

DOM (Document Object Model) представляет собой структуру документа в виде дерева объектов, где каждый узел представляет собой часть документа, такую как элемент, атрибут, текстовый узел и т.д.
Что делает метод createElement объекта document?

Метод createElement создает новый элемент HTML с указанным тегом.
Что делает метод document.body.append?

Метод document.body.append добавляет созданный элемент в конец тела (body) документа.
У каких объектов есть метод append?

Метод append доступен у объектов, представляющих DOM-элементы, таких как Element и DocumentFragment.
document и body - это объекты?

Да, document и body - это объекты. document представляет весь документ, а body представляет элемент <body> документа.
