Чем отличаются HTML и CSS?
HTML (HyperText Markup Language) является языком разметки и используется для создания структуры содержимого веб-страницы. Он определяет элементы, такие как заголовки, параграфы, изображения, ссылки и др.
CSS (Cascading Style Sheets) является языком стилей и используется для задания внешнего вида и оформления элементов HTML. Он определяет цвета, шрифты, отступы, расположение элементов и многое другое.
Для чего нам нужен тег style?
Тег <style> в HTML используется для внедрения стилей (CSS) непосредственно внутрь веб-страницы. Он позволяет определить визуальное оформление элементов на странице, применяя стили к ним.

Перечислите пару CSS свойств.
font-size: Устанавливает размер шрифта текста.
color: Задает цвет текста или фона элемента.
width: Устанавливает ширину элемента.
height: Устанавливает высоту элемента.
