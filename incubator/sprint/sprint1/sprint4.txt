Чем отличаются HTML/CSS/JS?
HTML - разметка для структуры веб-страницы.
CSS - язык стилей для оформления внешнего вида элементов.
JS - язык программирования для создания интерактивности на веб-страницах.

Что такое тег script?
Тег <script> в HTML используется для вставки JavaScript-кода в веб-страницу, обеспечивая выполнение скрипта в браузере.
