Что такое HTML?
HTML (HyperText Markup Language) - это язык разметки, используемый для создания структуры и содержания веб-страниц. Он состоит из различных тегов, каждый из которых представляет собой определенный элемент страницы.

Для чего нужен тег div?
Тег <div> используется в HTML для создания блочного контейнера, который позволяет группировать и стилизовать другие элементы внутри него. Он часто используется для разделения и организации содержимого веб-страницы.

Для чего нужен тег input?
Тег <input> используется для создания различных формовых элементов, таких как текстовые поля, кнопки, флажки и др. С атрибутами, такими как type, placeholder, value, он позволяет определить тип и поведение ввода.

Для чего нужен тег select?
Тег <select> создает выпадающий список, который позволяет пользователям выбирать один или несколько вариантов из предопределенного списка. Его внутри могут содержаться теги <option>, представляющие отдельные варианты выбора.

Для чего нужен тег img?
Тег <img> используется для вставки изображений на веб-страницу. Атрибут src указывает путь к изображению, которое следует отобразить.

Для чего нужен тег audio?
Тег <audio> позволяет встроить аудиофайлы на веб-страницу. С помощью атрибута src указывается путь к аудиофайлу, а с помощью атрибута controls добавляются элементы управления (панель воспроизведения).

Что такое атрибут тега? Какой одинаковый атрибут есть у тега img и audio?
Атрибут тега - это дополнительная информация, предоставляемая внутри тега для определения его свойств или поведения. У тегов <img> и <audio> общий атрибут src, который указывает на источник (путь) изображения или аудиофайла. Этот атрибут используется для определения ресурса, который должен быть отображен или воспроизведен.
